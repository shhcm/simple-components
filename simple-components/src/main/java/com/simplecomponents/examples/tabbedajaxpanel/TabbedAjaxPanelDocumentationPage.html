TabbedAjaxPanel Documentation

Create a div with a wicket id to which we can assign the TabbedAjaxPanel

                <!DOCTYPE html>
                <html xmlns:wicket="http://wicket.apache.org">
                    <body>
                        <!-- other content -->
                        <div wicket:id="tabbed-ajax-panel"/>
                        <!-- other content -->
                    </body>
                </html>

Create a list of Tab's and the TabbedAjaxPanel instance

                LinkedList<Tab> tabList = new LinkedList<Tab>();
                tabList.add(new Tab("tab 1", new PanelA(TabPanel.CONTENT_ID)));
                tabList.add(new Tab("tab 2", new PanelB(TabPanel.CONTENT_ID)));
                tabList.add(new Tab("tab 3", new PanelC(TabPanel.CONTENT_ID)));
                tabList.get(0).setHighlighted(true); // First tab should be highlighted.
                TabbedAjaxPanel tabbedAjaxPanel = new TabbedAjaxPanel("tabbed-ajax-panel", tabList);
                // You may want to run a JS function when tab content is
                // added via ajax, e.g. for initializing some tab content.
                tabbedAjaxPanel.setJavascriptOnTabLoaded("initAccordion();");
